# Machine Learning Fundamentals (CSCI 5521)

### Graduate Academic Project | University of Minnesota
* **Language/Stack:** MATLAB / Octave
* **Core Focus:** Classical Machine Learning, Statistical Pattern Recognition, and Optimization

---

### Project Overview & Scope
This repository hosts algorithmic implementations built from mathematical foundations to explore pattern recognition and predictive statistical modeling. Focus was placed heavily on programming the vector calculus and matrix operations underlying modern machine learning algorithms without the abstraction of high-level wrappers.

### Key Work Completed
* **Supervised Classifiers:** Built classical parametric and non-parametric decision architectures from scratch, mapping linear boundaries and maximum-likelihood parameters.
* **Dimensionality Reduction:** Implemented feature selection and feature projection transformations to map high-dimensional datasets to dense spaces.
* **Unsupervised Clustering:** Programmed expectation-maximization and centroid partitioning algorithms to uncover structural boundaries in unlabelled data.

### Skills & Technologies Applied
* **Mathematical Foundations:** Linear Algebra, Vector Calculus Optimization, Gradient Descents, Probability Distributions.
* **Algorithmic Tooling:** Linear Regression, Dimensionality Reduction, Classifiers, Matrix Decomposition.
* **Environments:** MATLAB mathematical computing language.
